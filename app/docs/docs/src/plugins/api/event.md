

|   插件译名解释   |     事件名称     |                           触发时机                           |
| :--------------: | :--------------: | :------------------------------------------------------: |
|  插件已加载  |  @plugin-loaded  |      当且仅当插件首次被加载，包括重载、卸载后重新加载等      |
|  窗口被显示  |   @plugin-show   | 当且仅当每次窗口显示时触发，例如从enabled状态变为active状态或由其它插件的intend/mini-intend跳转打开 |
|  插件被卸载  | @plugin-unloaded | 当且仅当插件每次被卸载时调用，将会取消加载插件并从 pluginList 从移除 |
